## 📌 Overview

SubPilot AI is an enterprise-grade SaaS Subscription and Spend Management platform designed to help individuals and organizations monitor subscriptions, reduce unnecessary expenses, forecast future costs, and automate renewal management.

The platform combines AI-powered analytics, subscription monitoring, payment tracking, renewal reminders, and financial insights into one modern dashboard.

# ⚡ Installation

Clone Repository

```bash
git clone https://github.com/rahulsamanta82/SubPilotAi.git
```

Go to Project

```bash
cd SubPilotAi
```

Install Dependencies

```bash
npm install
```

Run Development Server

```bash
npm run dev
```

Build Project

```bash
npm run build
```

Preview Production Build

```bash
npm run preview
```
